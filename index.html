Captive Portals Suck

Captive portals already sucked. But they suck even more when all the sites you visit use HTTPS, since they can't redirect those connections. The end result is the internet seems broken, because you never get redirected to the portal page, so you never click accept, so none of your connections go through.

So I made this site expressly for those situations, so that I'd have a non-HTTPS page to hit. Feel free to use it yourself too.
